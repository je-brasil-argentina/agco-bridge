## Prerequsites To build

1. [NPM](https://nodejs.org/es/download/)
2. [Android SDK](https://developer.android.com/studio/?hl=es-419)
3. Cordova `$ npm install -g cordova`

## To Build

1. Clone Project

2. `$ npm install`

3. `$ ionic cordova build android`
